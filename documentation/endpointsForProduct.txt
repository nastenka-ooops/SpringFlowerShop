1. Создание цветка:
   - Описание: Этот эндпоинт позволяет создать новый цветок в магазине.
   - Путь: POST /api/v1/product
   - Параметры:
     нет
   - RequestBody
     - name (строка, обязательный) - название цветка
     - price (число, обязательный) - цена цветка
     - height (число, обязательный) - высота цветка
   - Ответ в случае успеха:
     {
       "id": 1,
       "name": "Роза",
       "price": 10.99,
       "height": "50"
     }
     ```

2. Получение информации о цветке:
   - Описание: Этот эндпоинт позволяет получить информацию о конкретном цветке по его идентификатору.
   - Путь: GET /product/{product_id}
   - Параметры:
     - product_id (число, обязательный) - идентификатор цветка
   - Ответ в случае успеха:
     {
       "id": 1,
       "name": "Роза",
       "price": 10.99,
       "height": "50"
     }

3. Получение информации о всех цветах:
   - Описание: Этот эндпоинт позволяет получить информацию о всех существующих цветках.
   - Путь: GET /api/v1/products
   - Параметры:
     Нет
   - Ответ в случае успеха:
    [ {
       "id": 1,
       "name": "Роза",
       "price": 10.99,
       "height": "50"
     },
     {
       "id": 2,
       "name": "Тюльпан",
       "price": 5.99,
       "height": "30"
     }
]
4. Изменение данных о цветке:
   - Описание: Этот эндпоинт позволяет изменить данные о существующем цветке в магазине.
   - Путь: PUT /api/v1/product/{product_id}
   - Параметры:
     - product_id (число, обязательный) - идентификатор цветка
    - RequestBody
     - name (строка, обязательный) - название цветка
     - price (число, обязательный) - цена цветка
     - height (число, обязательный) - высота цветка
     - Ответ в случае успеха;
     {
       "id": 1,
       "name": "Роза",
       "price": 12.99,
       "height": "60"
     }
5. Удаление цветка:
   - Описание: Этот эндпоинт позволяет удалить цветок из магазине.
   - Путь: Delete /api/v1//products/{product_id}
   - Параметры:
     - product_id (число, обязательный) - идентификатор цветка
   - Ответ в случае успеха:
      Product deleted successfully
